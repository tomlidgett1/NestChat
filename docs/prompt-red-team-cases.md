# Prompt Red-Team Cases

Use these when evaluating prompt or routing changes that affect tone.

## Continuation

- `haha true`
- `wait what do you mean`
- `nah`
- `mmm maybe`
- `go on`
- `that's not really it`

## Emotion

- `I'm honestly pretty flat today`
- `I'm so excited`
- `this is doing my head in`
- `I feel like I'm messing everything up`

## Intent Inference

- `can you sort that`
- `what should I do`
- `is this bad`
- `write something better`
- `fix this`

## Tone Switching

- The user is joking, then suddenly serious.
- The user is sharp or annoyed.
- The user is vulnerable but still wants action.

## Over-Answer Traps

- `thoughts?`
- `worth it?`
- `is that normal?`
- `bad idea?`

## Assistant-Cliche Traps

- `help me reply`
- `can you explain this`
- `what do you reckon`
- `tell me straight`

## Re-Entry Moments

- `good morning`
- `morning`
- `what's up`
- `you around?`

## What To Check

- The reply feels like a continuation, not a reset.
- It does not default to a question when one is not needed.
- It does not start follow-up questions with `Want ...?`
- It uses memory lightly when helpful.
- It reads casual vs serious task weight correctly.
